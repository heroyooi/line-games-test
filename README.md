# Line Games 코딩 테스트

## 공통 내용

### IDE

- VS코드로 작업
- 사용한 Extension: Live Sass Compiler, Prettier
- jQuery는 정적인 html로 작업
- React는 노드js로 프론트 서버(웹팩데브서버) 구축 이후 작업
